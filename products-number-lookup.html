About the Service

What is Number Lookup?

Recent years have seen number lookup queries making an appearance in campaigns and programs based on mobile phone numbers. These queries allow companies to check the validity and state of mobile numbers in their databases, giving them the opportunity to optimize various processes and decrease costs by ensuring accurate, up-to-date mobile number information.

Technical Details

Understanding Phone Number Structure

Each phone number contains information that makes it unique from the rest of the world's phone numbers.

IMSI
International Mobile Subscriber Identity

15-digit unique identifier for mobile subscribers on GSM networks

MCC
Mobile Country Code

First 3 digits identifying the subscriber's home country

MNC
Mobile Network Code

2 digits identifying the mobile network operator

MSIN
Mobile Subscription ID

Last 10 digits uniquely identifying the subscriber

Complete 15-Digit Structure

In IMSIs, the first three digits represent the mobile network (MCC), the second two digits identify the country code (MNC), and the last ten digits are the MSIN - a 10-digit unique number that any carrier uses to identify a mobile phone.

HLR Lookup

Home Location Register (HLR) lookup queries the central database to check if a mobile number is active, which network it belongs to, and its current status. Essential for validating numbers before SMS campaigns.

  • Real-time number validation
  • Network operator identification
  • Number status verification
  • Reduce bounce rates

MNP Lookup

Mobile Number Portability (MNP) lookup identifies when subscribers have switched operators while keeping their number. Critical for routing messages to the correct network and avoiding delivery failures.

  • Detect ported numbers
  • Current operator identification
  • Optimize routing decisions
